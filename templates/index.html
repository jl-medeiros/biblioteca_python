BIBLIOTECA ANHANGUERA

Este projeto tem como objetivo principal a criação de um sistema de biblioteca virtual, desenvolvido como parte do Projeto Integrado do primeiro semestre do curso de Ciência de Dados. Através da implementação deste sistema, buscamos aplicar os conhecimentos adquiridos em banco de dados, modelando e gerenciando informações de forma eficiente. Além disso, o projeto visa explorar conceitos de desenvolvimento de software
